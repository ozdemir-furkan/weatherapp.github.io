Arama Sekmesi

Components Klasörü İçindeki Search Öğesindeki SearchBar.js Dosyasına input koduyla lokasyonun girileceği arama barını ekledim ve girilen lokasyonun hava durumu verilerinin alındığı apiyi fetch kodunu kullanarak kodlarım arasına ekledim. Girilen şehri yakalamak ve elde tutmak için React State içinde useState kullandım. useState kodu içerisinde searchValue ve bunu belirtmek için thisSearchValue parametresi oluşturdum. useState içindeki veriyi her tuşa bastığımızda değiştirmek için input içerisinde onChange kodunu kullandım. Klavyeden basılan tuşları almak için {(e)} içerisine alıyoruz ve parametrede belirtmek için thisSearchValue(e.target.value)} kullandım. Daha sonra arama barına girilen lokasyonu enter'a bastığımızda ekrana getirmesi için bir search parametresi oluşturdum ve enter'a bastığımda gelen veriyi jsona dönüştürüyorum ve gelen sonucu thisWeatherForecast(result) koduyla parametreme ekliyorum. App.js içinde props ettiğim thisWeatherForecast buraya geliyor.Gelen json sonuçları thisWeatherForecast adlı parametreme ekleniyor. Apiyi Çektiğim Site:https://openweathermap.org/

function SearchBar({thisWeatherForecast}){
const api={
key: "812f710ad356397eaf02b858d1d443c1",
base: "https://api.openweathermap.org/data/2.5/weather"
};
const [searchValue,thisSearchValue]= useState("");
const search = (e) =>{
if(e.key=="Enter"){
fetch(`${api.base}?q=${searchValue}&units=metric&lang=tr&appid=${api.key}`)
.then((data)=> data.json())
.then((result)=>{
thisSearchValue("");
thisWeatherForecast(result);
})
}
}
return(
<div className="search">
<input
className="search-input"
type="text"
placeholder="Şehir"
onChange={(e)=> thisSearchValue(e.target.value)}
value={searchValue}
onKeyPress={search}
/>
</div> );

Arka Planı Ayarlama

Background Klasörü içindeki index.js dosyasına arkaplan değiştirmesi için background isimli bir parametre oluşturdum. Klasörün içine verilerin alınacağı TakePicture.js dosyasını oluşturdum ve içerisine Pictures klasörü içerindeki resimleri importladım. Hava durumlarına göre ayrı ayrı resim kodları oluşturdum. TakePicture içerisinde havanın durumunu almak için situation parametresini oluşturdum. Havanın durumunu switch-case ile parametreye aktardım. situation parametresini index.js dosyasına exportlayarak arka plan ayarlama kodunu tamamladım. Daha sonrasında App.js dosyasındaki Background kısmına weatherforecast parametresinde bilgi var ise weatherforecast[0].main parametresini gönderiyoruz.

function index ({ background }) {
return(
<div className="background">
<div className="overlay"></div>
<TakePicture situation={background}/>
</div>
); } function TakePicture({situation}){
switch (situation){
